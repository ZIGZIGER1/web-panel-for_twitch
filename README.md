# mopsyan_sysan

Программа для стрима и OBS:

- `1.png` показывается в тишине
- `2.png` и `3.png` чередуются, когда ты говоришь
- есть отдельное окно для захвата в OBS
- можно выбрать зеленый chroma-фон или свою фоновую картинку
- можно включить прозрачный фон для OBS-окна
- есть локальная ссылка для `Browser Source` в OBS с настоящей прозрачностью
- есть вкладка `Помощь` с готовыми шагами для OBS и кнопками копирования
- в блоке чата есть кнопка `Вставить ссылку` из буфера
- у персонажа и UI есть более заметные анимации: idle-движение, bounce при речи и визуализатор звука
- можно вставить ссылку на чат и сделать `obs_chat_overlay.html`

## Запуск

```powershell
py -m pip install -r requirements.txt
py mopsyan_sysan.py
```

Или двойным кликом по `start_mopsyan_sysan.bat`.

Основной запуск теперь идет тихо: программа открывает web-панель и уходит в системный трей без черного окна консоли.
Тихий запуск идет через bootstrap и поднимает `pythonw`, чтобы tray-иконка Windows появлялась как GUI-приложение, а не как скрытый консольный процесс.

Для отладки можно отдельно запускать `start_mopsyan_web.bat`, если нужен старый консольный режим web-сервера.

## OBS

1. Во вкладке `OBS` выбери режим фона и размер сцены.
2. Если нужен прозрачный вывод, копируй `Browser Source URL` и добавляй его в OBS как `Browser Source`.
3. Если нужен chroma-фон, нажми `Открыть OBS-окно` и захватывай его как `Window Capture`, потом добавь `Chroma Key`.
4. Во вкладке `Помощь` есть готовая инструкция и кнопки `Копировать URL` и `Копировать инструкцию`.
5. Для чата вставь ссылку во вкладке `Чат`, нажми `Сделать HTML`, потом добавь этот файл в OBS.
