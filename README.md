# Web панель by ZigZiger

Локальная web-панель для стрима: персонаж, прозрачная сцена для OBS, кастомный Twitch-чат и тихий запуск в системный трей без старого ощущения от desktop-окна.

## Что умеет

- Живая web-панель управления со сценой, чатом, пресетами и быстрыми действиями для OBS.
- Прозрачный `Browser Source` для OBS через локальный URL.
- Персонаж с тремя кадрами (`1.png`, `2.png`, `3.png`) и реакцией на микрофон.
- Кастомный Twitch-чат с отдельным overlay, reveal-only режимом и стилями.
- Тихий запуск через `pythonw` с tray-иконкой вместо черной консоли.
- Сборка в `exe` и portable-архив для раздачи другим людям.

## Быстрый старт

### Вариант 1. Запуск из исходников

```powershell
py -m pip install -r requirements.txt
py mopsyan_sysan.py
```

Или двойным кликом по `start_mopsyan_sysan.bat`.

### Вариант 2. Готовый exe

После сборки готовый файл лежит в:

- `dist/Web_panel_by_ZigZiger.exe`
- `release/Web_panel_by_ZigZiger_portable.zip`

## Как это работает

- Основной запуск идет через `mopsyan_sysan.py`, который поднимает tray-оболочку.
- Локальный HTTP runtime живет в `web_runtime.py`.
- Интерфейс панели лежит в `web/index.html`, `web/styles.css` и `web/app.js`.
- Рендер сцены делает `scene_renderer.py`.
- Twitch-чат и overlay логика собраны в `chat_overlay.py`.

## Настройка OBS

1. Запусти панель и открой локальный адрес в браузере.
2. Возьми URL сцены из панели и добавь его в OBS как `Browser Source`.
3. Если нужен отдельный чат, добавь chat URL как второй `Browser Source`.
4. При необходимости настрой фон, позицию персонажа, момент эфира и стиль чата в панели.

## Настройка картинок персонажа

- `1.png` используется как idle-кадр.
- `2.png` и `3.png` чередуются во время речи.
- По умолчанию пути хранятся в portable-виде, чтобы проект можно было перенести на другой ПК без жесткой привязки к твоему `C:\Users\...`.

## Сборка exe

```powershell
py build_exe.py
```

Если `exe` уже собран и нужно только обновить portable-пакет:

```powershell
py build_exe.py --package-only
```

Быстрый bat-файл для сборки: `build_exe.bat`

## Структура проекта

- `web/` — интерфейс панели
- `assets/ui/` — текстуры и графика интерфейса
- `scene_renderer.py` — генерация кадра сцены
- `chat_overlay.py` — HTML/CSS/JS логика overlay-чата
- `web_runtime.py` — локальный сервер и состояние приложения
- `mopsyan_tray.py` — запуск и управление через системный трей
- `build_exe.py` — упаковка в `exe` и portable-release

## Репозиторий

В репозиторий специально не включаются:

- `build/`
- `dist/`
- `release/`
- `__pycache__/`
- локальные временные тестовые файлы

Это сделано для чистой публичной раздачи исходников.
